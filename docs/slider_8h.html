Detailed Description

Gestion des curseurs glissants pour réglages de valeurs.

Ce module fournit les structures et fonctions nécessaires pour créer et manipuler des curseurs glissants (sliders), principalement utilisés pour les réglages de volume et autres paramètres numériques.
