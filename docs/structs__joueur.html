Detailed Description

Référence au type joueur pour éviter les inclusions circulaires.

Représente le joueur contrôlé par l'utilisateur.

Contient toutes les propriétés du joueur: entité de base, contrôles, capacités spéciales, santé, inventaire, statistiques et équipement.
