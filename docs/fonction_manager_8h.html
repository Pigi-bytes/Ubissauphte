◆ FONCTION_PARAMS

#define FONCTION_PARAMS (   ...)    __VA_ARGS__, NULL

Ajoute automatiquement un NULL à la fin d'une liste d'arguments.

Parameters
...Liste d'arguments variadiques à passer à la fonction.
Returns
La liste des arguments fournie suivie de NULL.
Note
Évite d'ajouter manuellement NULL lors des appels de fonctions.

// Appel sans la macro exempleFonction("un", "deux", "trois", NULL);

// Appel avec la macro exempleFonction(FONCTION_PARAMS("un", "deux", "trois"));
